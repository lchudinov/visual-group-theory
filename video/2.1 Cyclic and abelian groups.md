# Cyclic and abelian groups

## Cyclic hroups

A group is **cyclic** if it can be generated by a single element.

Finite cyclic groups describe the symmetry of objects that have only rotational symmetry.

![Example](2.1%20Cyclic%20group%20example.png)

Generator: counterclockwise rotation by `2pi/n` (called a "click"), where `n` is the number of "arms". 

`Cn = <r, r^n = e>`

The order of a group G is the number of distinct elements in G, denoted by |G|.

The cyclic group of order n(i.e. n rotations) is denoted Cn (or sometimes Zn).

Zn comes from the fact that the binary operation for Cn is just modular addition.
To add two numbers in Zn, add them as integers, divide by n, and take the remainder.
For example, in Z6: `(3 + 5) mod 6 = 2` 

### Cyclic groups, additively

Group elements are written as 0, 1, 2, 3 ... n - 1
- 0 is the identity
- 1 is the single counterclockwise "click"



### Cyclic groups, multiplicatively

If r is a generator(e.g rotation by 2pi/n), then we can denote the n elements by
`1, r, r^2, ..., r^(n-1)`

r is a complex number e^(2pi/n), with a group operation being multiplication.
![Example](2.1%20Cyclic%20group%20and%20complex%20numbers.png)

r^n = 1, r^(n+1) = r, r^(n+2) = r^2, etc. - exponents represent modular division.

![Cayley diagrams](2.1%20Cayley%20diagrams%20examples%202.png)

Set Cn = {e^(2*pi*k/n) | k \in Z} under the binary operation of x.

## More on cyclic groups

The infinite cyclic group (additively) is (Z, +), the integers under addition.
Using multiplicative notation, the infinite cyclic group is
`G = <r | > = {r^k : k \in Z}`

For the infinite cyclic group (Z, +), only 1 or -1 can be generators.

Any number from {0, 1, ..., n - 1} that is relatively prime(взаимно простые) to n will generate Zn.

For example, 1 and 5 generate Z6, while 1,2,3, and 4 generate Z5.

Z6 = <1> = <5> Z5 = <1> = <2> = <3> = <4>

Z10 = <1> = <3> = <7> = <9>


A group is **abelian** if multiplication commutes.
