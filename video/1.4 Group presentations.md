## How to label nodes with actions

- Choose a node as our initial reference point; label it e. (This will correspond to our "identity action")
- Relabel each remaining node in the diagram with a path that leads there from node e. (If there is more than one path, pick any one; shorter is better)
- Distinguish arrows of the same type in some way (color them, label them, dashed vs solid, etc.)

`V_4 = <v,h, v^2 = e, h^2 = e, vh = hv>`

## Triangle puzzle group

`D_3`

![Triangle puzzle group](1.4%20Triangle%20puzzle%20group.png)

`G = <r, f>`
 - `r` - 120 degree rotation 
 - `f` - horizontal flip 

![Triangle puzzle group with labels](1.4%20Triangle%20puzzle%20group%20with%20labels.png)

`D_3 = <r,f | r^3 = e, f^2 = e, r^2f = fr>`
