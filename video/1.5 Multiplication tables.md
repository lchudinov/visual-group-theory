# 1.5 Multiplication tables

## Inverses

If `g` is a generator in a group, then the following the "g-arrow" backwards is an action
that we call inverse, and denoted by `g^(-1)`.

Note that by construction,
  `gg^(-1) = g^(-1)g = e`
